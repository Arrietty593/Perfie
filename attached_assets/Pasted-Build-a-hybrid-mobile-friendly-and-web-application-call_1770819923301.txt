Build a hybrid mobile-friendly and web application called Perfie — an AI-powered fragrance discovery platform with personalized recommendations, price comparison, wishlist, custom categories (MoodMix), and social login. Use modern full-stack architecture.

This app will allow users to:

Explore fragrance categories

Get personalized AI recommendations via Aro AI Assistant

Save favorites

Compare prices across external vendors

Create custom fragrance mood categories

Manage user profiles (self & others)

Tech Stack (Use these in the solution)

Frontend:

React (with Next.js for SSR & routing)

Tailwind / Chakra UI for responsive, mobile-friendly UI

Backend:

Node.js with Express or Fastify

RESTful API

Database:

PostgreSQL (preferred) OR MongoDB

Use Prisma ORM (if PostgreSQL) or Mongoose (if MongoDB)

Authentication:

JWT

OAuth: Google, Facebook, Instagram login

AI Integration:

Connect with an external LLM for Aro AI assistant (OpenAI API / similar)

Pricing & Sites Data:

Integrate scraping or partner API for real-time price comparison

Deployment:

Replit deployment + CI/CD
